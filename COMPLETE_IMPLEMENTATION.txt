💡 КЛЮЧЕВЫЕ ПАРАМЕТРЫ
═══════════════════════════════════════════════════════════════════════════════

transcribed_folder
  └─ Папка для обработанных видео
  └─ Тип: строка
  └─ Пример: "D:/Videos/Transcribed"

enable_speaker_identification
  └─ Определять ли спикеров
  └─ Тип: boolean
  └─ Значения: true / false
  └─ По умолчанию: false

enable_upload
  └─ Загружать ли видео на сервер
  └─ Тип: boolean
  └─ Значения: true / false
  └─ По умолчанию: true

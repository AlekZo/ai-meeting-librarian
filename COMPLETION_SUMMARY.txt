TROUBLESHOOTING
================================================================================

App won't start:
  1. Check logs: tail -f logs/auto_renamer.log
  2. Look for ERROR messages
  3. Fix the issue
  4. Delete app_state.json to reset: rm logs/app_state.json
  5. Restart

Files not processing:
  1. Check internet: ping 8.8.8.8
  2. Check Nextcloud status (should show green checkmarks)
  3. Check queue: cat logs/meeting_log_queue.json
  4. Restart app if needed

Too many errors:
  1. Check logs for actual error: tail -100 logs/auto_renamer.log | grep ERROR
  2. Fix root cause
  3. Restart app

See MIGRATION_GUIDE.md for more troubleshooting.
